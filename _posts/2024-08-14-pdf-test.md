## Features (All Native PDF.js)

- **Complete Navigation**: Page thumbnails, previous/next, jump to page
- **Advanced Search**: Full-text search with highlighting
- **Zoom Controls**: Fit to page, fit to width, custom zoom levels
- **View Modes**: Single page, scroll, spread view
- **Toolbar**: Complete PDF.js toolbar with all tools
- **Download & Print**: Native browser integration
- **Presentations**: Full-screen presentation mode
- **Annotations**: View PDF annotations and comments
- **Text Selection**: Copy text, view page info
- **Mobile Responsive**: Touch-friendly interface

## Usage

To embed a PDF using the official Mozilla PDF.js viewer, use:

```liquid
{% raw %}{% include pdf-viewer.html file="filename.pdf" title="Optional Title" caption="Optional caption" %}{% endraw %}
```

### Parameters:
- `file` (required): PDF filename in `/assets/pdfs/` directory
- `title` (optional): Custom title for accessibility
- `caption` (optional): Caption text below the viewer
