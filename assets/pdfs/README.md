# Sample PDF for Testing

This directory should contain your PDF files for embedding in blog posts.

## Example PDFs to Add

1. `sample.pdf` - A small test document
2. `user-manual.pdf` - Documentation
3. `research-paper.pdf` - Academic content
4. `presentation.pdf` - Slides or presentations

## File Naming Convention

- Use lowercase letters
- Use hyphens instead of spaces
- Be descriptive but concise
- Include version numbers when relevant

Examples:
- ✅ `user-guide-v2.pdf`
- ✅ `financial-report-2024.pdf`
- ✅ `api-documentation.pdf`
- ❌ `User Guide V2.pdf`
- ❌ `report final FINAL.pdf`

## File Size Recommendations

- **Small documents**: < 1MB (fast loading)
- **Medium documents**: 1-5MB (acceptable)
- **Large documents**: > 5MB (may load slowly)

For large files, consider:
1. Compressing images within the PDF
2. Reducing image resolution
3. Splitting into smaller documents
4. Using external hosting for very large files

## Security Note

PDFs are publicly accessible once placed in this directory. Do not include:
- Sensitive information
- Personal data
- Confidential documents
- Copyrighted material without permission
